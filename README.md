# generate-img

Generate image from mermaid diagram using mermaid-cli.

## Installation

```bash
npm install
```

## Usage

```bash
npm run generate
```

## Description

This project uses @mermaid-js/mermaid-cli to generate PNG images from mermaid diagram files (.mmd).

## Configuration

- `input`: `architecture.mmd` - your mermaid diagram file
- `output`: `architecture.png` - output PNG image
